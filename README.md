# Ecommerce Sales Dashboard (Power BI)

## Project Overview
This project is an interactive Power BI dashboard built to analyze ecommerce sales performance, customer behavior, and target achievement.

The dashboard provides insights into:

- Total Sales Performance
- Sales Target Achievement
- Customer Distribution
- Sales Trends Over Time
- Sales Team Performance
- Order Source Analysis

---

## Dashboard Pages

### 1. Overview
- Total Sales KPI
- Total Customers KPI
- Total Orders KPI
- Sales Target KPI
- Target Completion %
- Interactive filters

### 2. Sales Analysis
- Sales by Country
- Sales Trend by Month
- Sales by Sales Team
- Sales by Order Source

### 3. Target Analysis
- Target vs Actual Sales by Sales Manager
- Performance comparison
